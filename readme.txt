Plugin to add custom documentation to all admin pages.

== Description ==

With this plugin you can easily add custom documentation to each of the pages of the administration panel. This is very useful to improve the WordPress management helping your clients or your private users to find information related to the admin panel page they are managing.

== Installation ==

1. Install Custom Help either via the WordPress.org plugin repository or by uploading the files to your server.
2. Activate Custom Help.
3. Navigate to each of the pages where you want to add information and press the Wordpress help button.
4. Within the Wordpress help block you will have a new section to add customized information.
5. Click on Custom Help and use the form to add all the content you want using text, html or markdown.

== Frequently Asked Questions ==

= Who should use Custom Help? =

Custom Help is perfect for any user or company that wants to attach information to the information pages, either to remember it themselves or to help their clients or private users with the administration of Wordpress.

= Why markdown? =

With markdown you can easily format your content and enrich it, therefore it is the perfect solution for many users who haven't knowledge of html.
